# 📢 Бот-Зазывала для Telegram

Бот, который **упоминает всех участников чата** по команде — как @everyone в Discord!

## ✨ Главная функция

```
/call — зазвать ВСЕХ участников чата!
```

Все участники получат уведомление о том, что их упомянули.

## 🚀 Быстрый старт

### 1. Создайте бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям и получите токен бота

### 2. Настройте конфигурацию

```bash
# Скопируйте файл примера
cp .env.example .env

# Отредактируйте .env
nano .env
```

Заполните:
- `BOT_TOKEN` — токен от BotFather
- `ADMIN_ID` — ваш Telegram ID (узнать: [@userinfobot](https://t.me/userinfobot))

### 3. Установите зависимости

```bash
pip3 install -r requirements.txt
```

### 4. Запустите бота

```bash
python3 bot.py
```

### 5. Добавьте бота в чат

1. Добавьте бота в нужный чат
2. Назначьте его **администратором**
3. Дайте участникам поговорить (бот запоминает всех, кто пишет)
4. Используйте `/call` чтобы зазвать всех! 🎉

## 📋 Команды

| Команда | Описание |
|---------|----------|
| `/call` | 📢 **Зазвать всех участников** |
| `/call Ваш текст` | Зазвать с кастомным сообщением |
| `/зазвать` | То же самое на русском |
| `/all` / `/everyone` | Альтернативы на английском |
| `/stats` | Статистика бота |
| `/members` | Список участников в базе |
| `/help` | Справка |

## ❓ Как это работает

1. **Бот запоминает участников** — когда кто-то пишет в чат, бот сохраняет его в базу
2. **Админ вызывает /call** — бот формирует список всех участников
3. **Все получают уведомление** — бот отправляет сообщение с упоминанием каждого

## ⚠️ Важно

- **Команду `/call` могут использовать только администраторы** чата
- Бот должен быть **администратором** чата
- Бот собирает участников из сообщений — если человек не писал, его нет в базе

## ⚙️ Настройка сообщений

```python
CALL_MESSAGES = [
    "🔔 Внимание всем! Требуется ваше присутствие!",
    "📢 Всем привет! Загляните в чат!",
    # Добавьте свои сообщения...

```

## 🔧 Запуск в фоне (systemd)

Создайте файл `/etc/systemd/system/zazyval-bot.service`:

```ini
[Unit]
Description=Telegram Zazyvala Bot
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/path/to/bot
ExecStart=/usr/bin/python3 bot.py
Restart=always
RestartSec=10

[Install]
WantedBy=multi-user.target
```

```bash
sudo systemctl enable zazyval-bot
sudo systemctl start zazyval-bot
```
